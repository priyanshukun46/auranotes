# ✨ AuraNotes

> "Your brain is chaotic. AuraNotes isn't. 🧠✨"

AuraNotes is a clean, modern note-taking app built with **Ruby on Rails 8**. Whether you're dumping random shower thoughts, saving lecture notes, or planning your next billion-dollar startup, AuraNotes has your back.

---

## 🚀 Features

- 🔐 Secure Authentication (Devise)
- 📝 Create, Edit & Delete Notes
- 🎨 Clean UI with Tailwind CSS
- 📱 Responsive Design
- ⚡ Fast & Lightweight
- 🌙 Modern Minimal Design
- ❤️ Built with Rails because life is too short for boilerplate

---

## 🛠️ Tech Stack

| Tech | Why? |
|------|------|
| 💎 Ruby on Rails 8 | Backend Magic |
| 🎨 Tailwind CSS | Clean UI |
| 🔑 Devise | Authentication |
| 🗄️ SQLite / PostgreSQL | Database |
| ⚡ Turbo + Hotwire | No unnecessary JavaScript |

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/auranotes.git

cd auranotes

bundle install

bin/rails db:create
bin/rails db:migrate

bin/dev
```

Open

```
http://localhost:3000
```
